# Linux-tkg

My small personal Frankenstein-style fork for testing patches and configs.

This fork tracks [upstream](https://github.com/Frogging-Family/linux-tkg) closely and adds some spice on top.

## Extra knobs

All user-facing staging options are documented in
[`customization.cfg`](customization.cfg) under `STAGING OPTIONS`.

Leave an option empty to keep the normal prompt/default behavior. Set it in
`customization.cfg` or in your external config file to make builds deterministic.

Please do not report bugs from this fork to upstream.
